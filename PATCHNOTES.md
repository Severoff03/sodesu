## 0.10 — исправления интерфейса и сохранение файлов
- Исправлена критическая ошибка: при запуске главный экран не отрисовывался (пустые карточки/календарь). Причина — обращение к удалённым полям старой формы добавления библиотеки (`cType`/`cAdd`) в bindSettings → исключение прерывало инициализацию. Мёртвый код удалён.
- Инициализация сделана отказоустойчивой: сбой одного модуля больше не «роняет» весь интерфейс (каждый блок в защищённом вызове).
- Сохранение файлов (копия прогресса, экспорт библиотеки .csv) теперь через системный диалог с выбором места: добавлен нативный мост Android `saveFile` (Storage Access Framework) + веб-фолбэк.

## 0.10 — правки по списку
